The original aknabusiness.com used:
- Gold/charcoal color scheme

- Fire orange (#f97316) and amber (#f59e0b) gradients
- Playfair Display for headings, DM Sans for body

- Glowing gradient effects
- More modern card designs with rounded corners
- Animated floating elements
- Grid pattern backgrounds with diagonal lines
